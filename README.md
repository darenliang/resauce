# FileRenameTool

### Installation

Ubuntu:
```
sudo dnf install gtk3-devel glib2-devel
```

Fedora:
```
sudo dnf install gtk3-devel glib2-devel
```
Arch:
1. https://wiki.archlinux.org/index.php/Rust
2. https://wiki.archlinux.org/index.php/GTK/Development#Rust

OS X:
```
brew install gtk+3
```

Finally:
```
cargo run
```
